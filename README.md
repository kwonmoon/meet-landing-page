### What I learned

I made a mistake to define width and hight in image

Invalid
```html
<img
   src="./assets/desktop/image-man-texting.jpg"
   alt="picture of man texting"
   width: 510px
   height: 484px
>
```

Valid
```html
<img
   src="./assets/desktop/image-man-texting.jpg"
   alt="picture of man texting"
   width="510"
   height="484"
>
```
